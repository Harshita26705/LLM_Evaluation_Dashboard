What is LLM Evaluator?

An open-source platform for comprehensive LLM evaluation and analysis

LLM Evaluator is a powerful, easy-to-use platform designed to comprehensively evaluate Large Language Model responses. Whether you're a researcher fine-tuning models, a developer building AI applications, or an enthusiast exploring language models, our suite of evaluation tools provides deep insights into response quality, safety, fairness, and factuality.

Core Features

How to Get Started

Get LLM Evaluator running in minutes

Prerequisites

You'll need Python 3.8+ and pip installed on your system. All dependencies are automatically installed during setup.

1

Clone the Repository

Download LLM Evaluator from GitHub and navigate to the project directory.

git clone https://github.com/Harshita26705/LLM_Evaluation_Dashboard.git
cd LLM_Evaluation_Dashboard
2

Install Dependencies

Install all required Python packages using pip.

pip install -r requirements_flask.txt
3

Start the Server

Run the Flask application on your local machine.

python flask_app.py
4

Access the Dashboard

Open your browser and navigate to the application URL. Start evaluating!

http://localhost:5000
